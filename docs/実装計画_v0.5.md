> **最終更新:** 2026-04-13

## 完了済みフェーズ

### Phase 1: 作物拡張 ✅
- 作物15種追加（10種→25種+イベント2種=27種）
- 全作物の3Dボクセルモデル・色定義・レベル解放テーブル追加

### Phase 2: キャラクター + 畑拡張 ✅
- 人型キャラ統合（man/woman等 → human 1体、カラー変更で自由にカスタマイズ）
- 異形キャラ6体追加（犬, 猫, ロボット, エイリアン, カボチャ頭, 雪だるま）
- 実績によるキャラクター解放システム
- 複数畑スロット（最大3面、プレステージアップグレードで解放）
- プレステージ時の実績・コスメ・キャラ設定の保持修正
- プレステージショップのホバーツールチップ追加

### Phase 3: 自動アップデート + 季節イベント（v0.5.1）✅
- electron-updater + GitHub Releases による自動アップデート
- 季節イベント13種追加（春/夏/秋/冬）
- 季節限定3Dビジュアルエフェクト

### Phase 4: 図鑑システム（v0.5.2）✅
- 作物図鑑（累計収穫数、最高レベル、発見日、レア度表示）
- キャラクター図鑑（解放条件表示、ロック/アンロック状態）
- イベント図鑑（遭遇回数、季節フィルタ）
- コンプリート率（総合/作物/キャラ/イベント/実績）
- 既存の作物カタログ・イベント図鑑をタブ統合
- GameState に cropDiscoveredAt, cropHarvestCounts 追加

### Phase 5-1: 統計ダッシュボード（v0.5.3）✅
- 総プレイ時間計測
- 概要（プレイ時間、総収穫、累計ポイント、レベル、転生回数）
- 作物収穫ランキング（トップ5バー表示）
- プレステージ履歴（最新10件）
- GameState に totalPlayTime, prestigeHistory 追加

### Phase 5-2: 超越システム（v0.6.0）✅
- 超越条件: プレステージ20回 AND Lv.100
- 超越通貨 = floor(prestigeCount^1.2 × level/50)
- リセット範囲: ポイント, レベル, EXP, 種, 畑, プレステージ通貨/アップグレード
- 保持範囲: 超越データ, コスメ, 実績, 図鑑, 統計
- 超越アップグレード10種（基礎強化4種, プレステージ強化3種, 解放3種）
- 4つ目の畑、自動転生、黄金の種袋
- 不朽の遺産（超越後にプレステージアップグレードを一部保持）
- プレステージ通貨獲得量に超越ボーナス適用
- プレステージアップグレードコストに超越割引適用
- 超越回数に応じた称号（覚醒者/超越者/世界の理）
- 超越実績3種追加

## 技術的留意点

### 既存アーキテクチャとの整合性
- ES Modules + Vanilla JS パターンを維持（バンドラー不使用）
- Three.js CDN読み込みを維持
- localStorage セーブを維持（データ量増加に注意）
- 210×220px ウィンドウサイズの制約内でUI設計

### 後方互換性
- `loadState()` のマイグレーション処理で新フィールドをデフォルト値で補完
- `fieldState` → `fieldSlots` 移行済み
- 旧人型キャラID → `human` 自動変換済み
- 旧カタログ/イベント図鑑 → 図鑑モーダルへリダイレクト

### 超越システムの前提
- 超越はプレステージの上位リセット（プレステージ通貨もリセット）
- 不朽の遺産によりプレステージアップグレードを一部保持可能
- 自動転生はレベル閾値設定式（0で無効）
- prestigeHistory は最新20件に制限（localStorage容量対策）
