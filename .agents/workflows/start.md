# /start — 作業開始ワークフロー

// turbo-all

## 手順

1. **AImemory の確認**
   - `C:\Tools\AImemory\index.md` を読む
   - 現在の作業に関連するエントリがあれば `C:\Tools\AImemory\entries\` 内の該当ファイルを読む

2. **開発ルールの確認**
   - `docs/DEVELOPMENT_RULES.md` を読む

3. **プロジェクト構造の把握**
   - プロジェクトルートのディレクトリ構造を確認する
   - 主要な設定ファイル（`package.json`, `tsconfig.json`, `pyproject.toml` 等）を確認する

4. **要件の確認**
   - ユーザーの要件を正確に把握する
   - 不明点があれば質問する
   - 複数の実装方針がある場合は選択肢を提示する

5. **実装計画の提示**
   - 変更対象のファイルと変更内容を一覧で提示する
   - ユーザーの承認を得てから実装を開始する
